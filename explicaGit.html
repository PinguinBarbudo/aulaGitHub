Passo a passo

Primeira versão de um projeto GIT

Considerando que agora o seu ambiente já está todo configurado (usuário e email, visualização de arquivos ocultos, chave SSH), sempre que você criar um novo projeto, os passos básicos serão estes (troque os parâmetros em itálico pelos seus dados)

git init

git add .

git commit -m "mensagem explicativa"

git branch -M main

git remote add origin git@github.com:"seuusuario/seurepositorio".git

No caso específico da utilização do SSH do pinguinbarbudo

git remote add origin git@github.com:PinguinBarbudo/aulaGitHub.git

git push -u origin main
